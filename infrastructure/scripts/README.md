# Infrastructure Scripts

Utility scripts for setup, migrations, and maintenance.

## Available Scripts

| Script | Purpose | Status |
|--------|---------|--------|
| `setup-dev.sh` | Bootstrap local dev environment | Planned |
| `migrate.sh` | Run Prisma migrations on production | Planned |
| `seed-dev.sh` | Seed development database | Planned |
| `backup-db.sh` | Manual database backup trigger | Planned |

Scripts are created as needed. See `docs/runbooks/` for operational procedures.
